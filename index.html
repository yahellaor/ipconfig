🔎 מדריך: מציאת כתובת ה-IP הפנימית (LAN)

כתובת ה-IP הפנימית שלך נחוצה כדי לגשת להגדרות הנתב או להתחבר למכשירים מקומיים. כתובות אלו מתחילות בדרך כלל ב- 192.168, 172.16-31, או 10.0.0. הדרך המדויקת למצוא אותה היא באמצעות שורת הפקודה:

שלב 1: פתיחת שורת הפקודה (CMD)

  1. לחץ על כפתור התחל/Windows.
  2. הקלד CMD בשורת החיפוש.
  3. לחץ על התוצאה שורת הפקודה (Command Prompt) כדי לפתוח חלון שחור.

שלב 2: הפעלת הפקודה

ipconfig

הקלד את הפקודה ולחץ Enter.

שלב 3: זיהוי הכתובת הנכונה

חפש את הכותרת הרלוונטית לחיבור שלך:

  • 🔌 אם אתה מחובר עם כבל (קווי):
    חפש תחת Ethernet adapter.
  • 📶 אם אתה מחובר עם Wi-Fi (אלחוטי):
    חפש תחת Wireless LAN adapter Wi-Fi.

הכתובת שאתה מחפש היא הערך המופיע ליד IPv4 Address.

שלב 1: פתיחת הטרמינל (Terminal)

  1. לחץ על Command + Space כדי לפתוח את Spotlight.
  2. הקלד Terminal בשורת החיפוש.
  3. לחץ Enter כדי לפתוח את החלון השחור.

שלב 2: הפעלת הפקודה

ifconfig

הקלד את הפקודה ולחץ Return/Enter.

שלב 3: זיהוי הכתובת הנכונה

חפש את שם הממשק הרלוונטי:

  • 🔌 אם אתה מחובר עם כבל (קווי):
    חפש תחת en0 או en1.
  • 📶 אם אתה מחובר עם Wi-Fi (אלחוטי):
    חפש תחת en0, en1 או תחת השם של מתאם ה-Wi-Fi.

הכתובת שאתה מחפש היא הערך המופיע ליד inet תחת הממשק הנכון.

⚠️ הצגת הנתונים: המידע מוצג ישירות בדפדפן שלך. כתובות ה-IP הציבוריות נשלפות באמצעות פנייה לשירותי API ציבוריים, ואינן נשמרות או נשלחות לשום מקום אחר.

כתובת IP ציבורית (Public IP)

הכתובת הייחודית שספק האינטרנט שלך (ISP) מקצה לנתב שלך, המאפשרת למכשירים שלך לתקשר עם העולם החיצון (אתרים, שרתים). זו הכתובת שבה "רואים" אותך ברשת העולמית.

IPv4 (Internet Protocol version 4)

הגרסה הנפוצה ביותר של כתובות IP, המשתמשת בפורמט של ארבעה מספרים המופרדים בנקודות (למשל, 192.168.1.1). עקב מחסור בכתובות פנויות, היא מוחלפת בהדרגה על ידי IPv6.

IPv6 (Internet Protocol version 6)

הדור החדש של כתובות IP, המשתמש בפורמט ארוך יותר של ספרות ואותיות המופרדות בנקודותיים, ומציע כמות עצומה של כתובות (למשל, 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

כתובת IP פנימית (Private IP)

כתובת המשמשת לזיהוי מכשירים בתוך הרשת המקומית שלך (LAN). היא נחוצה לתקשורת פנימית ואינה גלויה מחוץ לבית או למשרד.

Default Gateway (שער ברירת מחדל)

זו כתובת ה-IP הפנימית של הנתב (Router) שלך. כל תעבורה שצריכה לצאת מהרשת המקומית שלך לאינטרנט עוברת דרך כתובת זו תחילה. בדרך כלל זו הכתובת הראשונה בטווח (כמו 192.168.1.1). לרוב ה-Gateway משמש גם כשרת DHCP.

DNS (Domain Name System)

המערכת המתרגמת שמות אתרים (כמו `google.com`) לכתובות IP מספריות (כמו `142.250.68.238`). זהו מנגנון קריטי שמאפשר לדפדפן שלך למצוא את השרת הנכון. 1.1.1.1 ו-8.8.8.8 הם שרתי DNS ציבוריים פופולריים.

VPN (Virtual Private Network)

רשת פרטית וירטואלית. VPN יוצר מנהרה מוצפנת בין המכשיר שלך לשרת מרוחק, ומנתב את כל התעבורה שלך דרכו. התוצאה היא שכתובת ה-IP הציבורית המוצגת היא הכתובת של שרת ה-VPN, מה שמקנה פרטיות ואנונימיות.

Latency (זמן השהיה / לאטנסי)

משך הזמן שלוקח לחבילת נתונים קטנה לעבור מהמכשיר שלך לשרת מרוחק ולחזור אליך (Ping). זמן השהיה נמוך (פחות מ-50ms) חיוני לחוויית גיימינג ושיחות וידאו חלקה.

Bandwidth (רוחב פס)

היכולת המקסימלית של חיבור הרשת שלך להעביר נתונים בפרק זמן נתון, הנמדדת במגה-ביטים לשנייה (Mbps). רוחב פס קובע את מהירות ההורדה והעלאה המקסימלית.
